Added double jump
added variable jumps_left set to 2

original jump logic:
Python

if self.jump_requested and self.on_ground:
    self.player_vel.y = -p.jump_speed
    self.on_ground = False

new jump logic:
Python
if self.jump_requested and self.jumps_left > 0:
    self.player_vel.y = -p.jump_speed
    self.on_ground = False
    self.jumps_left -= 1

Issue
Holding the jump button caused jumps_left to deplete to zero instantly because the game registered a jump on every frame.

The Solution
Introduced a should_jump flag to gate the jump trigger. This allows the player to either:

Press to jump normally.

Hold to automatically trigger the double-jump only once the character begins to fall (y velocity > 0).

Code Update
Before:

Python
if self.jump_requested and self.jumps_left > 0:
    self.player_vel.y = -p.jump_speed
    self.on_ground = False
    self.jumps_left -= 1
After:

Python
should_jump = self.jump_requested or (self.jump_held and self.player_vel.y > 0)

if should_jump and self.jumps_left > 0:
    self.player_vel.y = -p.jump_speed
    self.on_ground = False
    self.jumps_left -= 1
    self.jump_requested = False

self.jump_requested = False
